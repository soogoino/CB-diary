# 📋 問題改善建議

## 🔧 修改建議總覽

| 區塊 | 修改建議 |
|------|---------|
| **M7**（夜間勃起） | 0–10 次的 Slider 太細了。建議改為：「無 / 偶爾 / 頻繁」三個大按鈕，數據精準度對一般用戶來說沒那麼重要，體感才重要。 |
| **R 題庫** | 建議將 R 題庫改為「今日挑戰 (Daily Challenge)」。例如：今天 R5 是清潔，App 就跳出：「今天是清潔日！完成清潔並記錄可獲得額外積分。」 |
| **X 選填區** | 把它做成一個「+ 更多細節」的懸浮按鈕，而不是展開列表。只有強迫症或深度記錄者會點開它。 |
| **E1 / E6**（重複） | 兩題有點重複，建議合併成一個問題：「今天的心情狀態」，提供多選 Chips（快樂、焦慮、疲倦等）和一個額外的「其他」選項，讓用戶可以更靈活地表達當天的情緒。 |
| **E2**（照片） | 拍照確認後提供「隱私模糊」或「標籤遮蓋」功能，可以透過點擊圖片去除直接預覽，讓用戶在記錄時感到安全；觀看紀錄時預設為模糊。 |
